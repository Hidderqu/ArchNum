Notre langage de programmation est destine a afficher des elements sur un carte meteorologique animee.
Des elements peuvent etre crees puis animes dans le corps du programme ce qui aura pour consequence d'animer
l'affichage correspondant.

Toute la syntaxe du langage est decrite dans le fichier exemple "test0.met" dans des commentaires dont la syntaxe
est similaire a celle du langage C : /* Commentaire */

Pour tester la compilation du programme, il suffit d'exécuter la commande
python3.6 main.py [chemin d'accès au fichier test0.met]

Par exemple
python3.6 main.py ./test0.met
